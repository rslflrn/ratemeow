how to order

all books are secondhand. check availability on the catalogue page first.

to order, chat us on @rslflrn in this format:
- name
- phone
- social media [twitter / instagram / telegram]
- delivery address
- book code

admin will send a quotation. after payment is verified, we’ll deliver via tiki. we’ll update you with the book’s tracking code.

you can ask for a video of the book's condition first before purchasing.
